# agentkit-agents

Agent packages for [AgentKit](https://knew-inventai.github.io/agentkit/) — the Inventai AI tools marketplace.

Each directory is a standalone Claude Code agent installable via AgentKit.

## Structure

```
your-agent-name/
├── plugin.json    # AgentKit manifest
├── AGENT.md       # Claude Code agent definition
└── README.md      # optional usage docs
```
